# TaxClicker Pro 98

A SimCity-style tax town builder game for kids, styled like Windows 98.

## Play Now

**[Play TaxClicker Pro](https://timothyoverton.github.io/tax-clicker/taxclicker-pro.html)**

## About

Build and manage a small town. Collect taxes from buildings and citizens. Balance spending on services vs saving money. Watch your town grow!

- Target age: 6-10 years old
- Single HTML file, no install needed
- Windows 98 retro aesthetic
- Built iteratively by NS-TaxClicker agent
